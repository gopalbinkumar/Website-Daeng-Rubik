## 10. Navigation States & Interactions

### Active State
- **Current Page**: Bold text + underline atau background highlight
- **Color**: Accent color (merah/biru rubik)

### Hover State
- **Desktop**: 
  - Text color change
  - Underline animation
  - Background color (subtle)
- **Mobile**: 
  - Background highlight
  - Scale effect (minimal)

### Loading State
- **Navigation**: Skeleton loader untuk menu items
- **Breadcrumb**: Disabled state saat loading

### Error State
- **404 Page**: Navigation tetap accessible
- **Empty State**: Link ke halaman utama atau kategori lain

## 12. Accessibility Considerations

### Keyboard Navigation
- **Tab Order**: Logical flow dari kiri ke kanan, atas ke bawah
- **Focus Indicator**: Visible outline dengan warna kontras tinggi
- **Skip Links**: Link untuk skip ke konten utama

### Screen Reader Support
- **ARIA Labels**: Semua icon buttons memiliki label
- **Landmark Roles**: `<nav>`, `<main>`, `<footer>` dengan roles
- **Alt Text**: Semua gambar navigasi memiliki alt text

### Mobile Touch Targets
- **Minimum Size**: 44x44px untuk touch targets
- **Spacing**: Adequate spacing antara menu items
- **Feedback**: Visual feedback saat tap (ripple/highlight)
