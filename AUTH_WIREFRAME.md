## 10. RESPONSIVE BREAKPOINTS

### Desktop (1024px+)
- Split screen (40% visual, 60% form)
- Form card max-width: 480px
- Centered layout

### Tablet (768px - 1023px)
- Split screen (30% visual, 70% form)
- Form card: 90% width

### Mobile (<768px)
- Stacked layout
- Visual section: Small banner (120px height)
- Form: Full width dengan padding 20px
- Inputs: Touch-friendly (48px height minimum)
